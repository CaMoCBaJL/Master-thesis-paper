\begin{frame}
    \frametitle{Цель работы \slnumber}
    \Large
   Цель работы --- разработка  системы программирования роботов на основе плат Arduino на базе исходных кодов системы программирования КуМир.

\end{frame}


% Описание курсовой работы
\begin{frame}
    \frametitle{Актуальность работы \slnumber}
    \Large
    
    \begin{enumerate}
        \item снижение входного порога в области обучения робототехнике;
        \item применение навыков программирования в системе КуМир на практике;
        \item уменьшение затрат на приобретение программных и аппаратных средств разработки.
    \end{enumerate}

\end{frame}

\begin{frame}
    \frametitle{Постановка задачи \slnumber}
    \Large
   Опираясь на текущую кодовую базу среды программирования ``КуМир`` и дополнительный модуль трансляции языка программирования ``КуМир`` в языке С++, требуется разработать программный комплекс для изучения робототехники в школе с использованием языка программирования ``КуМир``.

\end{frame}


\begin{frame}
    \frametitle{План проекта  \slnumber}
    \Large
    Изучить и проанализировать:
	\begin{itemize}
        	\item исходные коды системы программирования КуМир;
        	\item историю развития технологий, используемых для разработки системы программирования;
        	\item системы программирования плат Arduino и их аналоги.
    \end{itemize}
\end{frame}

\begin{frame}
    \frametitle{План проекта  \slnumber}
    \Large
    Реализовать:
	\begin{itemize}
        	\item отдельный клиент для разработки роботов;
        	\item обновление используемых библиотек и фреймворков, используемых для разработки среды программирования;
        	\item доработки для устранения ошибок в исходном коде системы программирования КуМир;
        	\item плагин для взаимодействия с платами Arduino.
    \end{itemize}
\end{frame}

\begin{frame}
    \frametitle{План проекта  \slnumber}
    \Large
    По окончанию реализации и тестирования среды программирования ``КуМир роботы`` для разработки роботов, планируется спроектировать и разработать одного или нескольких роботов на базе аппаратного комплекса Arduino для опробации разработки в школах.
\end{frame}

\begin{frame}
    \frametitle{Этапы разработки программного комплекса  \slnumber}
    \Large
    \begin{enumerate}
    \item создание конфигурационного файла для генерации клиента ``КуМир роботы``;
    \item обновление зависимостей приложения на более современные версии;
    \item интеграция приложения Arduino-cli в систему программирования для отслеживания подключенных плат к компьютеру, прошивки, компиляции скетчей и обменом данными с платой в режиме реального времени;
    \item создание плагина-шаблонизатора плат для выбора .h-файла, содержащего набор предзаданных портов и констант;
    \end{enumerate}
\end{frame}

\begin{frame}
    \frametitle{Этапы разработки программного комплекса  \slnumber}
    \Large
    \begin{enumerate}
    \setcounter{enumi}{4}
    \item динамическое изменение статического анализатора при помощи внедренного шаблона платы;
	\item создание библиотеки шаблонов, позволяющей пользователям делиться шабонами роботов;
	\item интеграция с API github для повышения доступности библиотек шаблонов --- добавление возможности подключения в клиент репозитория с шаблонами;
    \end{enumerate}
\end{frame}
